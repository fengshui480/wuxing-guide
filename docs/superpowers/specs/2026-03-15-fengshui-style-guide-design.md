# 玄学形象指南 - 设计文档

## 产品定位

小红书商品橱窗售卖的付费网页测试产品（9.9元/次）。用户购买后获得网页链接，填写个人信息后生成个性化的"玄学形象指南"报告。

## 目标用户

小红书女性用户，25-35岁，对星座/玄学/穿搭感兴趣。

## 商业模式

- 渠道：小红书商品橱窗（虚拟商品）
- 定价：9.9元/次
- 交付：购买后发送网页链接
- 全部内容付费，无免费/付费分层

## 用户输入

| 字段 | 必填/选填 | 用途 |
|------|:-:|------|
| 性别 | 必填 | 妆容/穿搭方向 |
| 出生年月日 | 必填 | 天干地支、五行、生肖推算 |
| 城市 | 选填 | 个性化文案 |
| 行业/职业 | 选填 | 穿搭场景化建议 |

## 报告模块（从上到下）

### 1. 气场指数
- 综合评分（如 82/100）
- "超过X%同类型用户"对比文案
- 仪表盘式视觉展示

### 2. 气场关键词
- 3个词描述今日气质
- 示例："温柔锐利 / 低调奢华 / 暗涌玫瑰"

### 3. 今日开运色卡
- 主色 + 辅色 + 避忌色
- 色块可视化展示
- 附五行生克简要解释

### 4. 今日命定穿搭
- 场景化风格建议（通勤/休闲/约会）
- 开运配饰/材质子项
- 穿搭关键词

### 5. 今日开运妆容
- 口红色调 + 眼影色系 + 腮红方向
- 色块展示
- 与开运色联动

## 玄学逻辑链

```
出生日期 → 天干地支 → 日主五行 → 喜用神/忌神
                                      ↓
                        当日天干地支 + 喜用神生克关系
                                      ↓
                    ┌────────┬────────┬────────┐
                    颜色映射   材质映射   气质映射
                    (色卡)    (配饰)    (穿搭/妆容)
```

### 简化喜用神算法
1. 从八字中提取日主五行
2. 统计八字中各五行出现次数（天干+地支）
3. 判断日主强弱（得令/得地/得势）
4. 日主强 → 喜用神 = 克/泄/耗的五行
5. 日主弱 → 喜用神 = 生/助的五行

### 气运值计算
- 当日天干五行与喜用神的关系 → 基础分
- 当日地支五行与喜用神的关系 → 加减分
- 性别、生肖与当日冲合关系 → 微调
- 最终映射到 60-95 分区间（偏高分布，用户体验友好）

## 技术方案

- **架构**：纯前端单页应用（HTML/CSS/JS）
- **核心依赖**：lunar-javascript（via CDN），提供天干地支/八字/纳音/吉神方位等计算
- **部署**：GitHub Pages，零运维成本
- **无后端、无数据库、无用户系统**

## 视觉风格

现代轻奢 — 浅色渐变底（米白→淡金），高级灰+金色点缀，简洁卡片式排版。
- 报告截图友好：核心内容一屏展示
- 品牌水印 + 话题标签
- 字体：思源宋体（标题）+ 思源黑体（正文）
